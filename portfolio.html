iOS Projects for clients

Membit

Membit is a AR application in iOS that allows users to make memories using AR based on the location where they are taken. This allows users to have a more realistic view of past memories. We helped with their first large project in building the applicaiton at scale ready for Hudson Yards. We worked to get a new version on the applicaiton ready more new sponsorships and to get features out for Hudson Yards NYC. We used Swift to revamp the UI in the aepplication, help decide and impletment design and UX changes, use ARKit to fix certain AR bugs and build out new AR features.

Lover

We helped Lover build an initial MVP, a iOS app to help men with ED. To eliminate the need to go to the doctors and we all know Viagra doesn't work. We then built their main application, Lover. Lover is an iOS app that helps all sex, gender, and orientation find their sexual wellness and to better their sexlife. We used Swift to build the application, SceneKit for a bunch of 3D graphics in the application, used a CMS system and Python to build an API for the subscription model
